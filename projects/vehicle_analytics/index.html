Vehicle Traffic Analysis and Analytics System

In this project vehicles are detected from a surveillance video camera and then classified into HMV, LMV and SMV using Python and OpenCV API. YOLO algorithm is used for real time object detection.

Vehicle data is stored in a MySQL database. Analytics Dashboard is built using Flask and Google Charts are used to render graphs.

Technologies Used - Python Flask OpenCV Google Charts
